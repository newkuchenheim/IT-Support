Zentrale Verwaltung


Anleitung

  • Schritt 1: Wählen Sie Ihren Standort aus. Ggf. vorher auf Zeiterfassung klicken.
  • Schritt 2: Geben Sie Ihren Vor- und Nachnamen an.
  • Schritt 3: Wählen Sie die Antragsart aus.
  • Schritt 4: Wählen Sie den Grund aus. Bei Bedarf können Sie es erläutern.
  • Schritt 5: Geben Sie bitte mindestens ein Datum an.
  • Schritt 6: Klicken Sie auf dem Button "Send Mail". Es wird eine Outlook Vorlage geöffnet, die Sie dann versenden können.
